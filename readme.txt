== Description ==
[Woocommerce Order Quantity Control Min Max](https://#) plugin allows you to control  product's quantity for each order minimum and maximum . You can set your desired value minimum 
and maximum. One separate tab will create after plugin installation in product data field.



== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard
1. Click the upload tab
1. Browse for the plugin file (jploft-quantity-control-min-max) on your computer
1. Click "Install Now" and then hit the activate button
